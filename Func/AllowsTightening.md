# allowsTightening

```swift
VStack {
    Text("I'm always say I want change but I ain't same.")
        .foregroundColor(.red)
        .frame(width: 210, height: 60)
        .allowsTightening(true)
        
    Text("I'm always say I want change but I ain't same.")
        .frame(width: 210, height: 60)
        .foregroundColor(.blue)
        .allowsTightening(false)
}
```
