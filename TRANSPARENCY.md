# 透明度・表示制御

#### opacity

```swift
Text("Opacity 0.3").opacity(0.3)
Text("Opacity 0.7").opacity(0.6)
Text("Normal")
```

#### hidden

```swift
Text("Hidden").hidden()
Text("Visible")
```

#### disabled

```swift
Button("A") {}.disabled(true)
```

#### help

```swift
Text("Tap Me")
    .help(Text("Nothing happened."))
```
